📋 GAME RULES & NAVIGATION

🎮 How to Play

  • Use WASD keys to move around
  • Use Mouse to look around (click to enable)
  • Press SPACE to jump
  • Press E to interact with platforms

📚 Learning Journey

  • Complete modules in sequential order
  • Each module: 4 lessons + 1 simulation
  • Pass quizzes with 50%+ score
  • Complete all lessons to unlock simulation

⭐ Progression System

  • Earn XP for completing lessons
  • Level up every 1000 XP
  • Unlock achievements for milestones
  • Complete all 5 modules for certificate

🔒 Module Unlocking

  • Start with Empathy (unlocked)
  • Complete simulation to unlock next
  • Modules must be done in order
  • Review completed modules anytime
